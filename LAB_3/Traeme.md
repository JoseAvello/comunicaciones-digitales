# Laboratorio 3 - Modulación Pasabanda de Señales Binarias

Este repositorio contiene los archivos `.grc` desarrollados en GNU Radio Companion para el Laboratorio 3 de Comunicaciones Digitales.

El laboratorio se enfoca en la implementación y análisis de dos esquemas de modulación digital binaria:

- **OOK / ASK**: modulación por encendido y apagado de portadora.
- **FSK**: modulación por corrimiento de frecuencia.
